Rytmo 1.0.8

Build 17 · 2026-03-08

Highlights

  • The dashboard now reopens reliably from the Dock and the app menu without spawning or resurfacing the wrong window.
  • Closing the main dashboard keeps the app ready to restore the same window cleanly on the next launch or reopen action.

Fixes

  • Auxiliary windows such as Sparkle update prompts no longer replace the dashboard as the app's tracked main window.
  • Window restore behavior is more stable when the notch UI and background player are active at the same time.

Notes

  • This is a focused macOS stability update for window management and reopen behavior.
