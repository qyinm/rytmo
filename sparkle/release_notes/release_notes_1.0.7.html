Rytmo 1.0.7

Build 16 · 2026-03-07

Highlights

  • Completed tasks now disappear immediately from active task lists in the dashboard, menu bar, and notch views.
  • The home screen task sidebar keeps its position stable when you complete a task instead of jumping upward.

Fixes

  • Task completion no longer injects a synthetic due date that changed row height and caused layout shifts.
  • Task toggle behavior now uses the shared completion flow more consistently across compact task surfaces.

Notes

  • This release focuses on task list behavior and UI stability improvements.
